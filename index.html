Golden Sun Engine - HTML5 (MIT License)
Under development.

Prefer to use Chrome base browsers instead of Firefox.
The engine is not working properly on Firefox.

Use the same controls of VBA: arrows, Z, X, A, S, Backspace and Enter.
Press 1,2 and 3 for zoom.
Press M to unmute.
Double click on screen to fullscreen.
D for debug colision. F for fps. G for grid. K for keys debug. T for battle stats. L for sliders.
We have some maps in this demo, feel free to find them!
Use Q to cast Move. Use W to cast Frost. Use E to cast Growth (swap djinn first).
It can take a while to load. I'm not compressing the source for now.

See the source code on github.
